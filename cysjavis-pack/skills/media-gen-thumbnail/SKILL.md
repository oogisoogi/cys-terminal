# media-gen-thumbnail

썸네일은 클릭률을 좌우하는 별도 산출물이다. 영상 주제·톤에서 시선을 끄는 이미지를 만들고
짧고 굵은 텍스트를 얹는다.

## 절차

1. **콘셉트** → 검증: 영상 핵심 메시지에서 썸네일 후크(한 장면·대비·표정·키워드)를 정한다.
   유튜브 비율(16:9, 1280×720 이상) 타깃.
2. **이미지 생성** → 검증: `[[media-gen-image]]`로 배경/주피사체 생성(프롬프트 재작성 활용).
   필요하면 `[[media-gen-edit]]`로 인물·요소 합성(정체성 보존).
3. **텍스트 오버레이** → 검증: 3~5단어의 굵은 텍스트, 고대비·외곽선, 모바일 가독성. 텍스트는
   **대본 사실과 모순되지 않게**(과장 클릭베이트 자제 — 평판).
4. **A/B(선택)** → 검증: 후크가 다른 2~3안을 만들어 비교 제시.
5. **비용** → 검증: 유료 생성이면 `[[cost-preview-confirm]]`.

## 출력 계약

`media/thumbnail.<ext>`(+ A/B면 `thumbnail-a/b`) + 매니페스트(콘셉트·프롬프트·실비용). 상위
`[[media-gen]]`로 반환 — 영상 파일과 함께 업로드 산출물. 사실 모순·저가독 썸네일은 보고·수정.
