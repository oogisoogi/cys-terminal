# video-verify-audio-sync

소리와 그림이 어긋나면 즉시 티가 난다. 립싱크·오디오 연속을 검증한다.

## 점검 항목

- **립싱크**: 아바타 입 움직임이 발화와 맞는가(전·중·후반 샘플 구간에서 입↔음성 정렬 확인).
- **오디오 연속**: 청크/클립 경계에서 끊김·중복·튐·무음 구멍이 없는가(`audio/seg` 합성
  경계와 `avatar/clip` concat 경계).
- **레벨 일관**: 구간 간 음량이 급변하지 않는가(라우드니스 비교).
- **A/V 드리프트**: 영상 후반으로 갈수록 오디오가 점점 밀리지 않는가(누적 싱크 오차).

## 절차

1. **오디오 추출·정렬** → 검증: 최종 영상에서 오디오 분리, `audio/timestamps.json`과 대조.
2. **립싱크 샘플** → 검증: 발화가 많은 구간 몇 곳에서 입↔음성 정렬을 프레임으로 확인.
3. **경계·드리프트 점검** → 검증: 클립/청크 경계와 영상 전체 길이에서 끊김·드리프트 측정.
4. **집계** → 검증: 전부 통과면 GO. 아니면 NO_GO + 어긋난 구간·원인(합성/아바타/음성).

## 출력 계약

`{gate: "audio_sync", verdict: GO|NO_GO, issues: [{timecode, type, evidence}]}`. 상위
`[[video-verify]]`로 반환. NO_GO면 원인 단계(`[[video-stitch]]`·`[[heygen-avatar-render]]`·
`[[voice-clone-elevenlabs]]`)로 회송.
