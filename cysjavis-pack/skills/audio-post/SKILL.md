# audio-post

나레이션만 있는 영상은 밋밋하다. 음악 베드가 깔리고 나레이션 아래로 덕킹돼야 "프로페셔널"
하게 들린다. 이 스킬은 음악을 마련하고 최종 믹스를 만든다(나레이션 자체는 `[[voice-clone-elevenlabs]]`).

## 하위 스킬 오케스트레이션

1. `[[audio-post-music]]` — 톤에 맞는 음악 베드 선택 또는 생성(라이선스/저작권 안전).
2. `[[audio-post-mix]]` — 나레이션 아래 덕킹·레벨 정규화·페이드, 라우드니스 타깃.

## 절차

1. **톤 결정** → 검증: 영상 톤(에너지·장르·템포)을 대본/패키징에서 도출. 무드 1줄 확정.
2. **음악 마련** → 검증: `[[audio-post-music]]` — 베드 확보. **저작권 안전 소스만**(생성
   음악·로열티프리·보유 라이선스). 출처·라이선스 기록.
3. **믹스** → 검증: `[[audio-post-mix]]` — 음악을 나레이션 아래로 덕킹, 구간 페이드,
   유튜브 라우드니스 타깃(약 -14 LUFS 통합) 정규화.

## 출력 계약

`audio/music-bed.mp3`(라이선스 메타 포함) + `audio/mix-spec.json`(덕킹·레벨·페이드 파라미터).
상위 `[[youtube-video-pipeline]]`로 반환 → `[[video-stitch]]` 마스터 렌더에서 합쳐진다.
저작권 불명 음악은 사용 금지(평판·클레임 리스크).
